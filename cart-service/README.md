# Cart Service

This is a minimal skeleton for the cart microservice.

Structure:
- pom.xml
- src/main/java/com/ecommerce/cartservice/
  - CartServiceApplication.java
  - controller/
  - dto/
  - entity/
  - repository/
  - service/
- src/main/resources/application.yml

Next steps:

- Wire persistent datasource (MySQL or H2) in `application.yml` and add required dependencies.
- Add DTO validation and exception handling as needed.
